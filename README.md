# Agentic AI Weekly Research Agent

An advanced LangGraph research agent that tracks developments in agentic AI. Uses semantic chunking, multi-pass search, and a quality-scoring retry loop to generate research reports and LinkedIn post drafts.

## Architecture

The agent runs a 6-node pipeline built with LangGraph:

```
[Planner] → [Researcher] → [Quality Checker] ──→ [Report Writer] → [LinkedIn Drafter] → END
                                   │                     ↑
                                   │                     │
                                   └── (score < 7) ──────┘
                                        retry with
                                        refined queries
```

**Planner** — Decomposes the query into 3-5 sub-questions with targeted search queries. Mode-aware: adjusts framing for weekly, monthly, or custom research.

**Researcher** — Runs multiple targeted searches per sub-question using Tavily. Fetches full page content from top results, then uses OpenAI embeddings and cosine similarity to extract the most relevant ~10k characters per page. On retry, reads quality feedback and searches for different, higher-quality sources.

**Quality Checker** — Uses GPT-4o to evaluate research quality on a 1-10 scale. Rates source credibility, flags unsupported claims, identifies contradictions, and notes gaps. Scoring calibrates to query type (overview, technical, or news).

**Report Writer** — Synthesizes findings into a structured Markdown report with inline citations, limitations section, and source list.

**LinkedIn Drafter** — Extracts key facts from the report and formats them as a concise LinkedIn post with a `[PERSONAL_INSERT]` placeholder for the author's voice.

## Key Features

- **Semantic chunking**: Pages are split into ~500 character chunks, embedded, and ranked by cosine similarity to the query. Only the most relevant content reaches the LLM.
- **Multi-pass search**: Each sub-question triggers 3 targeted searches instead of one broad query, covering different angles and source types.
- **Quality retry loop**: If research scores below 7/10, the agent loops back to the Researcher with the Quality Checker's feedback. After 3 retries, it proceeds with unverified items clearly noted.
- **Mode-aware**: Weekly mode searches the past 7 days. Monthly mode covers 30 days. Custom mode handles any query with no date restriction.
- **Versioned reports**: Multiple runs on the same day auto-increment filenames (report.md, report_v2.md, report_v3.md). No reports are ever overwritten.

## Setup

### Prerequisites
- Python 3.10+
- OpenAI API key
- Tavily API key (free tier: 1,000 searches/month at https://tavily.com)

### Installation

```bash
git clone https://github.com/radfordalex/agentic-ai-weekly.git
cd agentic-ai-weekly
python -m venv .venv

# Windows
.venv\Scripts\activate

# Mac/Linux
source .venv/bin/activate

pip install langgraph langchain-openai langchain-tavily langchain-community python-dotenv httpx numpy langchain-text-splitters
```

### Configuration

Create a `.env` file in the project root:

```
OPENAI_API_KEY=your-openai-key
TAVILY_API_KEY=your-tavily-key
```

## Usage

### Weekly Mode
Search the past 7 days for new developments:
```bash
python main.py weekly
```

### Monthly Mode
Summarize the past 30 days of trends:
```bash
python main.py monthly
```

### Custom Mode
Run any research query:
```bash
python main.py custom "What is agentic AI and what is the current state of production agentic AI systems?"
```

Reports save to `reports/` with date-stamped filenames.

## File Structure

```
agentic-ai-weekly/
├── state.py       # ResearchState TypedDict — shared state flowing through the graph
├── tools.py       # Tavily search + semantic page fetching with embeddings
├── agents.py      # 5 node functions (planner, researcher, quality_checker, report_writer, linkedin_drafter)
├── graph.py       # Graph wiring: nodes, edges, conditional retry loop
├── main.py        # Entry point with mode selection and versioned report saving
├── reports/       # Generated reports and LinkedIn drafts
└── .env           # API keys (not committed)
```

## Tech Stack

- **Orchestration**: LangGraph (graph-based state machine)
- **LLMs**: GPT-4o-mini (planner, researcher, report writer, LinkedIn drafter), GPT-4o (quality checker)
- **Search**: Tavily Search API
- **Embeddings**: OpenAI text-embedding-3-small
- **Semantic chunking**: LangChain RecursiveCharacterTextSplitter + cosine similarity ranking

## Cost Per Run

Approximate cost using OpenAI pricing:
- GPT-4o-mini nodes: ~$0.02-0.05
- GPT-4o quality checker: ~$0.10-0.20
- Embeddings: ~$0.01
- **Total: ~$0.15-0.30 per run** (higher with retries)
